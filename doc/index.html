Prerequisites

  • NodeJS v16.13.1
  • yarn v1.22.17 (or npm)
  • Vite Local server and building tool

Dependencies

  • Recharts
  • Prettier
  • EsLint
  • styled-components
  • prop-types

1 | Back-end

  1. Download or clone the back-end's repository :
git clone https://github.com/WARDI-Kamal/P9-front-end-dashboard.git
  1. Install dependencies :
# Yarn
yarn install
  1. Launch back-end on port 3000 (default port) :
# Yarn
yarn dev

2 | Front-end

  1. Clone this repository :

git clone https://github.com/WARDI-Kamal/WARDI-Kamal_12_16022022.git

  1. Install dependencies:
# Yarn
yarn install
  1. Launch dev server on port 3000 :
# Yarn
yarn dev

  1. Front-end is now rendered at URL http://localhost:3001 with mocked data. If you want to switch to real backend data, go to src/services/Global.js (this file) and set ENV constant from dev to prod.

  2. You also can build a production version and serve it with a static server [OPTIONNAL] :

# Yarn
yarn build
  1. If you need to install Vite : Link to documentation
